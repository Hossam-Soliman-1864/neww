# Footer
Footer is apython library for dealing with word pluralization.
## Installation
Use the package manager [pip](http:www.google.com) to install foobar.
> pip install foobar

## Usage
> import foobar
>
> foobar.pluralize('word') #returns 'word'\
foobar.pluralize('goose') #returns 'geese'\
foobar.pluralize('word') #returns 'word'
